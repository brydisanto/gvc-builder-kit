# GVC Community Images

152 images and GIFs from the Good Vibes Club universe, organized for easy use in your projects.

## How to use

Copy any image into your project's `public/` folder, then reference it in your code:

```tsx
<img src="/craig-landing.gif" alt="Craig" />
```

Or with Next.js:

```tsx
import Image from "next/image";
<Image src="/craig-landing.gif" alt="Craig" width={400} height={400} />
```
