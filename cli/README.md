# create-gvc-app

**The Playground** -- A builder toolkit for the Good Vibes Club community.

Go from idea to live project in minutes. No coding experience needed. The GVC brand system is baked in, Claude knows the design rules, and deploying is one command.

## Quick Start

```
npx create-gvc-app
```

That's it. The CLI walks you through everything:

1. Name your project
2. Pick what you want to build
3. Describe your idea
4. Choose extras (blockchain data, badges, leaderboards, etc.)

Your project is ready with the full GVC brand system, a Claude guide, and everything you need to start building.

## Commands

| Command | What it does |
|---|---|
| `npx create-gvc-app` | Create a new project (no install needed) |
| `npm run dev` | Start your project locally |
| Deploy via [Vercel](https://vercel.com) | Ship it live to the internet |

**Want shortcuts?** Install globally with `npm install -g create-gvc-app` to unlock these:

| Command | What it does |
|---|---|
| `gvc dev` | Same as `npm run dev` |
| `gvc deploy` | Deploy to Vercel from the terminal |
| `gvc templates` | Browse all available templates |

## What's Inside Every Project

**GVC Brand System**
- Gold (#FFE048), black, dark grays
- Brice + Mundial fonts
- Gold glow effects, shimmer animations, glassmorphism cards

**Claude-Ready**
- Every project includes a CLAUDE.md that teaches Claude the GVC brand
- Open your project in Claude and start building in plain English

**GVC Community API**
- Zero-config access to live collection data (floor price, holders, sales)
- Badge leaderboard with all 101 GVC badges
- VIBESTR token data and history

**Badge System**
- Full badge engine for offline evaluation
- 101 badge definitions with images and requirements
- Badge-token mapping for all 6,969 GVCs

- **GVC Collection data** -- live listings, floor price, owner data

- **Web3 wallet connect** -- RainbowKit + wagmi
- **Badge collection** -- all 101 badges with tier glow effects
- **Leaderboard** -- daily, weekly, all-time rankings
- **Stats panel** -- animated counters and charts
- **Game engine** -- scoring, daily challenges, game logic
- And more
